重新设计一个现代化的**智能运维大脑**（AIOps Dashboard），我们需要脱离传统“静态图表堆叠”的旧模式，打造一套以 **“事件驱动（Event-Driven） + Agent 辅助决策 + 人机协作（Human-in-the-Loop）”** 为核心的新一代 React 界面架构。

界面整体建议采用 **主次分明的“三栏驾驶舱”布局**，配合高对比度的暗黑主题（Dark Mode），兼顾全局态势与微观诊断。

## 二、 核心功能模块设计细则

### 1. 顶部 Header：态势感知与 Agent 状态

* **系统综合健康度 (Health Index)**：通过算法将 CPU/内存、Sentry 报错率、SQS 消息积压等维度收敛为一个 0~100 的健康分（如 `98.5%`），附带趋势曲线。
* **Agent 状态心跳**：显示 Agent 巡检进度（如：`🟢 AI Agent 正在实时巡检 (已连续运行 48 小时，捕获并收敛 12 次异常)`）。

---

### 2. 左栏：可观测性数据源与下钻 (Telemetry Panel)

不堆叠全量图表，而是作为**诊断证据库**：

* **数据源健康态指示灯**：实时显示 Sentry、Fluentd、Prometheus、Alertmanager 等数据源的连通性。
* **微型趋势图 (Sparklines)**：列出关键指标（如 API 5xx 比例、SQS 死信队列积压数）。点击任意指标，直接在中间底部的“下钻面板”展开完整的 Recharts 图表。

---

### 3. 中间主工作区：智能诊断与根因分析 (Core Workbench)

这里是“大脑”输出核心价值的地方，建议分为三个核心视图：

#### 视图 A：告警智能收敛 (Alert Noise Reduction)

* 当 Alertmanager 抛出 100 条告警时，AI 自动进行语义与时序聚类，**将其收敛为 1 个 Incident（事件卡片）**。
* 展示告警压缩率（例如：`已自动收敛 85% 冗余告警`）。

#### 视图 B：根因分析拓扑图 (RCA Node Topology)

* 使用 **React Flow** 渲染交互式拓扑图。
* **链条可视化**：`Sentry 前端 500` $\rightarrow$ `SQS 消息消费延迟` $\rightarrow$ `Redis 连接池溢出 (根因节点高亮闪烁)`。
* 点击任意节点，左栏或底栏会自动同步弹出该节点的完整 Log 和 Trace 堆栈。

#### 视图 C：自愈与优化方案 (Self-Healing Actions)

* 展示 AI 算出的建议操作（例如：“建议扩容 SQS 消费端 Pod 副本数自 2 至 5” 或 “清理占用 8000 端口的死锁进程”）。

---

### 4. 右栏：AI Copilot 人机协作区 (Agentic Assistant)

这是 Agent 的交互灵魂，需体现 **可解释性（Explainability）** 与 **安全可控性（Control）**：

* **推理过程展示 (Reasoning Chain)**：折叠展示 AI 的思考过程（如 *`[1/3] 正在查询 Sentry Stack Trace...`* $\rightarrow$ *`[2/3] 检索 AWS SQS 监控指标...`*）。
* **Tool Call 交互卡片**：AI 调用外部 API（如查询 Prometheus、发送飞书通知）时，呈现动态卡片。
* **人机确认机制 (Human-in-the-Loop Card)**：
对重启服务、放回死信队列（Redrive）、变更配置等高风险操作，弹出独立卡片：
> ⚠️ **高风险自愈建议**：清空 Redis 异常 Cache 并重启 Pod `orders-v2-xxx`
> **预估影响**：短暂闪断 1.5 秒，解决 95% 的请求死锁
> `[ 拒绝 ]` `[ 授权 Agent 执行 ]`

## 四、 核心 UI/UX 体验细节小技巧

1. **证据链附着（Evidence Attachment）**：
AI 给出的每一条分析结论（如“*由于 21:00 部署导致数据库连接溢出*”），句末都要带上可点击的 `[查看 Sentry 日志]` 或 `[查看 Git Commit]` 标签，点击直接高亮对应日志行。
2. **渐进式披露（Progressive Disclosure）**：
默认只给**结论和操作建议**，隐藏复杂的 JSON 报文和几千行的 Log；运维工程师点击“深入排查”时再逐层展开。
3. **快捷键盘响应**：
支持 `Cmd + K` 或 `Ctrl + K` 调出 Agent 快捷全局指令框，方便老运维直接敲命令控制 Agent（例如：`/analyze incident-123`）。
